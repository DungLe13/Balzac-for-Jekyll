National Football League on Twitter

by Dung Le

Project Description

National Football League on Twitter is a project that illustrates how Twitter's users feel and react before, during and after the football match between the New England Patriots and the New York Jets. The football match lasted for more than an hour, from 1:00 pm EST on Sunday, October 15th to 2:20 pm on the same day. The data is streamed using Twitter API from Sunday, October 15th 12:50 pm EST to Monday, October 16th 1:20 am EST. Only tweets with geodata tagged are presented in the map.
